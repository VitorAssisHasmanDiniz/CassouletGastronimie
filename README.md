# Cassoulet Gastronomie
## Sobre
Esse é um projeto acadêmico requisitado pelo professor Jean da Fatec Jenssen Vidal para uma universidade Fictícia. Por gostos pessoais, escolhi uma universidade de gastronomia, a Cassoulet Gastronomie.

## Funcionalidades
### Home
A home possui uma foto fictícia da faculdade, junto com o sobre da faculdade e os cursos oferecidos. Assim como todas as paginas, ela possui uma navbar com os links de navegação e um rodapé com os direitos autorais da universidade.

### Quem Somos
A pagina Quem Somos possui quem é a Cassoulet, motivos para escolher essa faculdade entre as outras e os objetivos da faculdade.

### Contatos
Esta pagina tem como objetivo realizar o contato entre o futuro aluno ou veteranos e a oordenação da Cassoulet. Possui um formulário objetivo, requisitando o email, o assunto do contato e a descrição do contato.
